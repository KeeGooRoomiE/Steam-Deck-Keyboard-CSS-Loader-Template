# Customizable Keyboard — Plugin Package

This folder contains the ready‑to‑install version of the Customizable Keyboard theme for Steam Deck’s CSS Loader.

Drop this folder into:
```
~/homebrew/themes/
```
and the theme will appear in Decky → CSS Loader.

---

## Included
- `theme.json` — CSS Loader manifest with built‑in color controls and rounding presets.
- `keyboard.css` — main stylesheet.

---

## Install
1. Copy this entire folder to `~/homebrew/themes/`
2. Open Decky → **CSS Loader**
3. Enable **Customizable Keyboard**
4. Press **Refresh Themes** if needed
